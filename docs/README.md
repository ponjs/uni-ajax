# 介绍

由于 [uni.request][1]在实际项目开发中存在一些不方便，自己在 Vue 项目中习惯用 [Axios][2]作为请求库。参考`Axios`的使用方式封装`uni.request`，但更轻量，且保留`uni.request`的核心。在`uni-app`项目开发中切合实际，开箱即用，配置简单。

::: tip
如有什么问题可以在 [插件市场][3]上评论留言，源码注释清晰，可以的话在 [Github][4]点个 star 吧！
:::

### 特性

- 支持 Promise API
- 支持 Typescript 开发
- 拦截请求和响应
- 自定义配置请求实例
- 多种 Method 方法请求
- 支持 RequestTask 操作
